# mysql80_no_usui_hon

## MySQL 8.0の薄い本とは？

 - MySQL 8.0で導入された新機能をページ数の制約（および著者の能力）の範囲でできるだけ取り上げるとともに、紹介記事のURLを提供するものです。
 - この本では従来のMySQLについて丁寧な説明は行いません。
 - はじめてMySQLに触れる方は、まずMySQL 5.7までの入門書・解説書などを読んでMySQLを実際に起動・操作し、全体像を掴んでおくことをお勧めします。
